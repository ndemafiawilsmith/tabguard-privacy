# TabGuard Privacy Policy

TabGuard is a browser extension designed to protect sensitive tabs using local PIN-based locking.

## Data Collection
TabGuard does **not** collect, transmit, or store personal data on external servers.

## Local Storage
All data is stored **locally on the user's device**, including:
- Encrypted PIN hash
- Locked domain preferences
- Session timeout settings

## No Tracking
TabGuard:
- Does not use analytics
- Does not use cookies
- Does not communicate with third-party services

## Permissions
Permissions are used strictly for functionality:
- Tabs: to detect active tabs
- Storage: to save settings locally
- Idle: to auto-lock when inactive
- WebNavigation: to detect page loads
- Alarms: to manage session timeouts
